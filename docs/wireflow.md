### wireflow   
  
  
``` 
[앱 첫 실행]
   ↓
[시작하기]
   ↓
[로그인] ───→ [회원가입] ──→ [본인인증] ──→ [로그인]
   ↓                ↑
[비밀번호 찾기] ───┘

[홈 화면]
   ├─ [검색창/검색 버튼] ──→ [검색 결과 페이지]
   │                              └─ [거래글 상세 페이지]
   ├─ [카테고리] ─────────→ [카테고리별 거래글 목록]
   │                              └─ [거래글 상세 페이지]
   ├─ [추천도서] ────────→ [추천 도서 상세 페이지] ─→ [대여하기] → [검색 결과 페이지]
   ├─ 하단바
   │   ├─ [홈화면]
   │   ├─ [추천도서]
   │   ├─ [커뮤니티] ──→ [글쓰기 페이지]
   │   │                      └─ 업로드 후 → [커뮤니티]
   │   ├─ [채팅 목록] ──→ [채팅방]
   │   │                      ├─ [약속 잡기] → [거래 관리 페이지]
   │   │                      ├─ [거래관리] → [거래 관리 페이지]
   │   │                      └─ [+] 캘린더, 사진, 송금
   │   └─ [마이페이지] ──→ [프로필 수정]
   │                          ├─ [결제수단 관리]
   │                          ├─ [찜한 글 목록]
   │                          ├─ [빌린 책 목록] ─→ [매너평가]
   │                          ├─ [등록한 책 목록] ─→ [거래 관리 페이지]
   │                          ├─ [내 커뮤니티 글 목록]
   │                          └─ [팔로워/팔로잉 조회]
```
